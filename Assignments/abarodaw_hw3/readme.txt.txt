# DESCRIPTION:
Program to find forward kinematics of a Franka Emika Panda cobot using DH parameters

# ENVIRONMENT USED:
JetBrains PyCharm Community Edition


# LIBRARIES USED:
Install the following libraries through the Pycharm IDE. Type the library name and if the IDE doesn't detect it, there will be a yellow line below the library name. Hover over the yellow line and option will pop up to install the library/package.

1. sympy
2. pprint

# GETTING STARTED:
Install the necessary libraries to have the code set up for running. The code outputs each transformation matrix and the final transformation matrix from 0th to 7th frame. Follow the below step to run the code.

# RUNNING THE CODE:
Click the play button on the top right corner or use the shortcut (Shift+F10).
